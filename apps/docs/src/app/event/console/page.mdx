# 웹 콘솔에서 Event 관리

### 개요

Event 등록·조회·수정·삭제는 DataGSM 웹 콘솔에서 수행합니다. 별도의 API Key나 OAuth 토큰 없이, DataGSM 웹에 로그인한 계정 권한으로 본인이 등록한 Event를 관리합니다.

### Event 페이지 진입

DataGSM 웹에 로그인한 뒤 상단 네비게이션의 **Event** 메뉴로 이동합니다. Event 페이지에서 현재 계정에 등록된 Event 목록을 확인할 수 있습니다.

### Event 등록

1. Event 페이지 우측 상단의 **추가** 버튼을 클릭합니다.
2. 등록 폼에서 다음 항목을 입력합니다.

| 항목         | 설명                                                            |
|------------|---------------------------------------------------------------|
| **수신 URL** | 이벤트를 수신할 외부 서버의 URL. `http://` 또는 `https://`로 시작해야 합니다.        |
| **구독 이벤트** | 이 Event로 전달받을 이벤트를 체크박스로 선택합니다. 최소 1개 이상 선택해야 합니다.            |

3. **추가** 버튼을 눌러 등록을 완료합니다.

> 평문 노출 위험이 있으므로 가능하면 `https://` 엔드포인트를 등록하는 것을 권장합니다. 구독할 수 있는 이벤트 종류는 [이벤트 페이로드 명세](/event/payloads)를 참고하세요.

발급된 `secret`은 외부 수신 서버가 `X-DataGSM-Signature` 헤더를 검증할 때 HMAC-SHA256 서명 키로 사용됩니다. 자세한 내용은 [서명 검증 가이드](/event/signature)를 참고하세요.

### Event 목록 조회

Event 페이지에 진입하면 현재 계정에 등록된 Event가 목록으로 표시됩니다. 각 항목에는 수신 URL, 구독 이벤트, 활성화 여부, 생성 일시가 표시됩니다.

> 보안상 `secret`은 목록에 표시되지 않습니다. 등록 시 노출된 값을 별도로 보관해 두세요.

### Event 수정

1. 목록에서 수정할 Event의 **편집(연필)** 아이콘을 클릭합니다.
2. 수신 URL 또는 구독 이벤트를 변경한 뒤 **저장**을 누르고, 확인 다이얼로그에서 다시 **저장**합니다.

### Event 삭제

목록에서 삭제할 Event의 삭제 동작을 실행합니다. 삭제된 Event는 복구할 수 없으며, 이후 발생하는 이벤트도 더 이상 전송되지 않습니다.

### 등록 가능 개수

하나의 계정에는 최대 **10개**의 Event를 등록할 수 있습니다. 이미 10개를 등록한 경우 등록 버튼이 비활성화되며, 새 Event를 등록하려면 기존 Event를 먼저 삭제해야 합니다.
